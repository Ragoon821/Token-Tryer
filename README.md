A Python-based tool to validate Discord tokens efficiently. TokenTryer allows users to check tokens either from a file or by direct input, with a user-friendly console interface and colorful output.

**Developed by: Ragoon821**

## Overview

TokenTryer is a lightweight script designed to help users validate Discord tokens by interacting with Discord's API. It supports checking multiple tokens from a file or a single token via manual input. The tool features a visually appealing console menu, making it easy to navigate and use.

## Features

- **Check Tokens from File**: Load and validate multiple tokens from a `tokens.txt` file, stopping when a valid token is found.
- **Check Single Token**: Manually enter a single token to verify its validity instantly.
- **GitHub Link**: Quick access to the developer's GitHub page (`github.com/ragoon821`) directly from the menu.
- **Fancy Interface**: Includes an ASCII art banner and color-coded output for better readability (powered by `colorama`).
- **User-Friendly Menu**: Simple console menu to select options and keep the application running until manually exited.
